### Requirements
- Spigot or PaperSpigot (1.12.2 or higher).
- [DataBaseInterface](https://github.com/andrei1058/DataBaseInterface) (Required for player data).
- [PlaceholderAPI](https://www.spigotmc.org/resources/placeholderapi.6245/) (Optional but recommended).

### Installation
1. Download the plugin from the official source.
2. Place the `.jar` file in your server's `plugins/` directory.
3. Restart the server.
4. Configure your database settings in `plugins/DataBaseInterface/config.yml`.
5. Restart again to ensure connection is established.

### Setting Up Arenas
1. Use `/ss create <name>` to start creating an arena.
2. Follow the on-screen instructions or use `/ss setup` for more options.
3. Set up the lobby location with `/ss setlobby`.
4. Define the tasks, vents, and sabotage points on the map.
5. Save the arena when finished.
