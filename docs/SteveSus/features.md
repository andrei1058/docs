### Tasks
Tasks are the core mechanic for crewmates. They are randomly assigned to players and can be configured per map.
* **Long Tasks**: Multi-stage tasks that take longer to complete.
* **Short Tasks**: Quick tasks that can be done in one step.
* **Common Tasks**: Tasks that all crewmates share.
* **Visual Tasks**: Tasks that show a visual effect when completed, allowing others to verify someone is a crewmate.

### Sabotages
Impostors can trigger sabotages to create chaos and separate crewmates.
* **Oxygen (O2)**: Must be fixed before the timer runs out, or impostors win.
* **Reactor Melt**: Must be fixed by two players at the same time.
* **Lights**: Reduces vision for all crewmates until fixed.
* **Comms**: Disables the task list and security cameras.

### Deployment Modes
SteveSus can be deployed in three ways:
1. **MULTI_ARENA**: Multiple arenas on a single server, with a central lobby.
2. **BUNGEE**: Uses a connector plugin to distribute players across multiple arena servers dynamically.
3. **BUNGEE_LEGACY**: One arena per server, ideal for simple setups.

### Per-Player Language
The plugin supports multiple languages. Players can select their preferred language, and all messages, titles, and menus will be displayed in that language.

### Anti-Abandon System
To prevent players from leaving the game early, SteveSus includes a penalty system for those who disconnect while the game is in progress.
