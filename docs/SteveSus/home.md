### Introduction
SteveSus is a Minecraft mini-game inspired from the famous "impostor" game and "murder mystery" mini-game. This plugin is meant to be used on spigot or paperspigot, other forks are not officially supported!

### Gameplay Overview
The default gameplay is based on two teams: **crewmates** and **impostors**.
- **Impostors** should sabotage the spaceship (or equivalent) and/or kill crewmates without getting caught in order to win.
- **Crewmates** have to complete tasks and pay attention to other teammates to find out who is faking tasks and identify the impostors.

### Main features:
* **🧹tasks for crewmates** (long / common / short and visual): Tasks are randomly assigned to players. You can set how many tasks you want on a map and install addons for extra tasks.
* **🧨sabotages for impostors**: Sabotages can only be used by impostors. They have configurable cooldowns, and some require immediate fixing.
* **👳‍♂️per player language**: Each player receives messages, titles, and GUIs in their selected language.
* **🌌PAPI support**: Placeholders can be used in almost every part of the plugin.
* **🚀Flexible deployment**:
  * **MULTI_ARENA**: One server with a lobby and multiple arenas.
  * **BUNGEE**: One server with multiple arenas (auto-scale), players join via *SteveSusConnector*.
  * **BUNGEE_LEGACY**: One arena per server instance.
* **⭐Rich API**: Developers can add new tasks, sabotages, teams, custom color assigners, and more.
* 🔝**Player stats**: Includes an anti-abandon system to ensure fair competition.
* 🍕**Parties support**: Supports internal and external party plugins like Parties by AlessioDP.
* 🍱**Rich customization**: Customize arena selector layouts, voting GUIs, all messages, and game settings.
