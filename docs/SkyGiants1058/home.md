# SkyGiants1058

SkyGiants1058 is an advanced Minecraft minigame where teams battle to protect their Giant while trying to destroy the enemy's Giant. It features a unique setup UX, dynamic localization, and multiple game modes.

### Key Features
- **Multiple Modes**: Supports MULTIARENA, NETWORK (Bungee/Velocity), and SMP modes.
- **In-game Setup UX**: Context-aware tools that change based on your location (Team base or Beast area).
- **Region Visualization**: Particle-based visualization for team and Beast regions (RECTANGLE and OVAL shapes).
- **Dynamic Localization**: Automatic language switching for sidebars, GUIs, and items.
- **Web Events**: Support for Redis and RabbitMQ to stream game events.
- **Achievements & Stats**: Comprehensive tracking for kills, wins, giant damage, and more.
